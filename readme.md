# 🎉 Event Booking System

A simple web-based application for browsing, booking, and managing event registrations.

## 🛠️ Features

- User registration & login
- Book events
- View Booked Events based on User Login

---

## 🧰 Tech Stack

- PHP (COdeigniter)
- MySQL
- HTML/CSS/JavaScript (Frontend)
- Bootstrap(UI)
